# hola-mundo-en-java

## Descripción

Programa "hola mundo" en Java

## Compilación y ejecución del programa

Para **compilar** el programa teclea lo siguiente (es necesario tener el *JDK*):

```console
javac HolaMundo.java
```

Y para **ejecutarlo**:

```console
java HolaMundo
```
